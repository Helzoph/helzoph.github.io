3 防火墙

TeamSpeak3 需要用到这些端口，请注意防火墙的设置

服务 协议 本地端口 必要?
语音 UDP 9987 是
文件传输 TCP 30033 是
ServerQuery (raw) TCP 10011 可选
ServerQuery (SSH) TCP 10022 可选
WebQuery (http) TCP 10080 可选
WebQuery (https) TCP 10443 可选
TSDNS TCP 41144 可选

下面以 iptables 防火墙为例

  • -A INPUT 添加 INPUT 链的防火墙规则
  • -p udp 指定 UDP 协议
    • 同理 -p tcp 指定 TCP 协议
  • -dport 9987 指定目标端口
  • -j ACCEPT 匹配规则时执行的动作
    • ACCEPT 代表同意
    • REJECT 代表拒绝
    • DROP 代表丢弃
